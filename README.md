Data Analysis workflow for ligation based Mass Spec signal enhancement of nucleic acids

Pytheas scripts are used with minor adjustments from [D'Ascenzo et al. Nature Communications (2022)](https://github.com/ldascenzo/pytheas).
File 4_label.py adds user specified label to database to search for ligation products. RNA that has not undergone ligation is maintained in the search data base

R scripts for data visualization primarily rely on the ggplot2 package.
